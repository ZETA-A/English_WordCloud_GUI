# WCG(WordCloudGui) 프로그램
## 만든 사유 : 영어공부하려고

### 제작자 : 제타
### 사용한 언어(?) : Python3(3.8.8에서 테스트됨) / JSON
### 사용법 :
1. 디버깅합니다
2. 세팅해줍니다(사실 기본세팅 되어있어서 세팅할 것도 없습니다)

### 주의사항 :
1. matlab라이브러리때문에 포팅이 안됩니다
2. 직접 main.py를 다운받고 디버깅하시길 바랍니다
3. 만약 포팅에 성공하신다면 머지신청부탁드립니다
